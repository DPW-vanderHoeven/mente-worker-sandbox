# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this repository,
please report it through GitHub's private vulnerability reporting:

- Go to the **Security** tab of this repository and click
  **Report a vulnerability**, or visit
  https://github.com/mente/mente-worker-sandbox/security/advisories/new

Please do **not** open a public issue or pull request for security reports.

Include as much of the following as possible:

- A description of the issue and its impact
- Steps to reproduce (proof-of-concept, if available)
- Affected versions, commits, or files
- Any suggested mitigations

## Response

We aim to acknowledge reports within 5 business days and to provide a
resolution or mitigation plan within 30 days. Reporters will be credited in
the resulting advisory unless they request otherwise.

## Scope

This policy covers the code in this repository. Reports about third-party
dependencies should be directed to the respective upstream project.
